StingFit PWA

Nainštaluj StingFit

StingFit funguje ako lokálna PWA aplikácia na telefóne aj desktope. Bez účtu, cloudu a telemetrie. Dáta zostávajú v tomto prehliadači, kým ich výslovne neexportuješ.

Otvoriť StingFit tréning

iOS Safari

  1. Otvor StingFit URL v Safari.
  2. Ťukni na Share.
  3. Vyber Add to Home Screen.
  4. Potvrď názov StingFit a otvor aplikáciu z plochy.

Android Chrome

  1. Otvor StingFit URL v Chrome.
  2. Ak sa zobrazí výzva, vyber Install.
  3. Ak nie, otvor menu a vyber Install app alebo Add to Home screen.
  4. Po inštalácii otvor StingFit z ikony aplikácie.

Desktop Chrome/Edge

  1. Otvor StingFit URL v Chrome alebo Edge.
  2. Použi install ikonu v adresnom riadku alebo menu prehliadača.
  3. Potvrď inštaláciu a pripni aplikáciu, ak ju chceš mať pri plánovaní po ruke.

Offline poznámka

Otvor aplikáciu aspoň raz online. Service worker potom drží lokálny app shell, /#/training a offline fallback pripravený pre ďalší tréning.
